# P1：封面

## 图类型：封面型

大标题居中，副标题在下方，底部留出信息区域。整体大气简洁，一眼抓住注意力。

## 约束

- 所有文字必须使用中文
- 不要出现任何英文字母（阿拉伯数字不算英文）
- 不要出现照片写实元素
- 不要出现真实白板的边框
- 不要出现白板笔
- 不要出现白板笔架

## 风格

干净的白色白板背景，类似 Excalidraw 的手绘草图风格，粗彩色马克笔笔触，简单的涂鸦图标，歪歪扭扭的手绘箭头。

## 字体

所有文字统一使用"站酷快乐体"风格——圆润、活泼、有手写感但清晰可辨。具体要求：
- 标题：粗体手写，字号最大
- 正文/标签：常规手写，字号中等
- 备注/页脚：手写，字号最小
- 全图只用一种手写字体，不混用其他字体风格

## 提示词

```
一张白板手绘风格的封面图，作为介绍深圳的第一页。

【主标题区域】（位于画面正中偏上，占整体高度约 40%）：
- 超大号粗体手写标题"介绍深圳"
- 标题左侧画一个摩天大楼简笔画（代表现代都市），右侧画一个渔船简笔画（代表历史起源）
- 标题文字用蓝色马克笔，加粗，是整张图最醒目的元素

【副标题区域】（主标题正下方，占整体高度约 15%）：
- 橙色手写中号字："从小渔村到世界级创新之都"
- 副标题下方画一条手绘波浪线作为装饰分隔

【装饰区域】（画面中间散布，占整体高度约 30%，不遮挡文字）：
- 左下角：一个手绘的城市天际线简笔画，高低错落的楼群
- 右下角：一个手绘的地图标记图标，里面写着"深圳"
- 标题周围散布几个小星星和上升箭头装饰，用不同颜色（蓝、绿、橙）

【底部信息区域】（位于图片最底部，占整体高度约 15%）：
- 居中，灰色小号手写："粤港澳大湾区 · 核心引擎"
- 底部画一条淡灰色手绘横线作为页脚分隔
```
