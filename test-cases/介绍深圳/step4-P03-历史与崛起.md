# P3：历史与崛起

## 图类型：时间线型（Timeline）

从左到右展示深圳 40 年发展里程碑，突出"奇迹般的速度"。

## 约束

- 所有文字必须使用中文
- 不要出现任何英文字母（阿拉伯数字不算英文）
- 不要出现照片写实元素
- 不要出现真实白板的边框
- 不要出现白板笔
- 不要出现白板笔架

## 风格

干净的白色白板背景，类似 Excalidraw 的手绘草图风格，粗彩色马克笔笔触，简单的涂鸦图标，歪歪扭扭的手绘箭头。

## 字体

所有文字统一使用"站酷快乐体"风格——圆润、活泼、有手写感但清晰可辨。具体要求：
- 标题：粗体手写，字号最大
- 正文/标签：常规手写，字号中等
- 备注/页脚：手写，字号最小
- 全图只用一种手写字体，不混用其他字体风格

## 提示词

```
一张白板手绘风格的图，展示深圳 40 年发展历程。

【PPT 标题区域】（位于图片最顶部，占整体高度约 12%，与下方内容有明显分隔）：
- 左侧：页码标记"02"，用灰色小号手写
- 中间：大号粗体标题"40 年奇迹"
- 标题下方画一条手绘横线作为分隔

【引导语】（标题下方，内容区上方）：
- 蓝色手写："从人口不足 3 万的小渔村，到全国第三的国际都市"

【内容区域】（占整体高度约 65%，横向时间线布局）：

画一条粗手绘横向箭头，从左到右贯穿整个画面，箭头上方标注年份节点：

节点 1（最左，灰色）：
- 圆形节点标注"1980 前"
```
