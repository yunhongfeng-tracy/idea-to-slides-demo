# Step 4：布局与格式展示（图片生成提示词）

每页一个提示词，用于白板手绘风格图片生成。以下为通用约束和风格，适用于全部 8 页。

## 通用约束

- 所有文字必须使用中文
- 不要出现任何英文字母（阿拉伯数字不算英文）
- 不要出现照片写实元素
- 不要出现真实白板的边框、白板笔、白板笔架

## 通用风格

干净的白色白板背景，类似 Excalidraw 的手绘草图风格，粗彩色马克笔笔触，简单的涂鸦图标，歪歪扭扭的手绘箭头。

## 通用字体

所有文字统一使用"站酷快乐体"风格——圆润、活泼、有手写感但清晰可辨。具体要求：
- 标题：粗体手写，字号最大
- 正文/标签：常规手写，字号中等
- 备注/页脚：手写，字号最小
- 全图只用一种手写字体，不混用其他字体风格

## 通用生成参数

- 宽高比：16:9
- 尺寸：4K

## P01：封面

### 图类型：封面型

大标题居中，副标题在下方，整体大气简洁。

### 提示词

```
一张白板手绘风格的封面图，作为"介绍深圳"PPT 的第一页。

【主标题区域】（位于画面正中偏上，占整体高度约 40%）：
- 超大号粗体手写标题"介绍深圳"
- 标题左侧画一个摩天大楼简笔画（代表现代化），右侧画一个海浪简笔画（代表滨海城市）
- 标题文字用蓝色马克笔，加粗，是整张图最醒目的元素

【副标题区域】（主标题正下方，占整体高度约 15%）：
- 橙色手写中号字："从小渔村到世界级创新之都"
- 副标题下方画一条手绘波浪线作为装饰分隔

【装饰区域】（画面中间散布，占整体高度约 30%，不遮挡文字）：
- 左下角：一个手绘的地图简笔画，标注"广东"和"深圳"位置
- 右下角：一个手绘的火箭简笔画，正在起飞（代表飞速发展）
- 标题周围散布几个小星星和小圆点装饰，用不同颜色（蓝、绿、橙）

【底部信息区域】（位于图片最底部，占整体高度约 15%）：
- 居中，灰色小号手写："粤港澳大湾区核心城市"
- 底部画一条淡灰色手绘横线作为页脚分隔
```

## P02：深圳概览

### 图类型：中心辐射型

深圳在中心，关键数据围绕四周辐射排列。

### 提示词

```
一张白板手绘风格的图，介绍深圳的基本概况。

【标题区域】（位于图片最顶部，占整体高度约 12%）：
- 左侧：页码标记"02"，用灰色小号手写
- 中间：大号粗体标题"深圳在哪里？"
- 标题下方画一条手绘横线作为分隔

【内容区域】（标题区下方，占整体高度约 75%）：

中心位置：
- 画一个手绘的中国地图轮廓简笔画（简化版），用红色圆点标注深圳位置
- 深圳位置旁边用蓝色大字标注"深圳"
- 地图右侧紧贴画一个小方块标注"香港"，中间画一条虚线表示相邻

围绕地图的四个信息气泡（用虚线圆角方框，分别在左上、右上、左下、右下）：
- 左上（蓝色框）：画一个尺子图标，写"面积：1997 平方公里"
- 右上（绿色框）：画一个人群图标，写"人口：1766 万"

- 右下（紫色框）：画一个奖牌图标，写"中国四大一线城市"

从每个气泡画一条歪歪扭扭的虚线箭头指向中心地图。

【底部区域】（占整体高度约 13%）：
- 居中蓝色手写："粤港澳大湾区核心城市，紧邻香港"
- 下方画一条淡灰色手绘横线
```
