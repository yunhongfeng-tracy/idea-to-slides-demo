# 想法变成图片 · 从一句话到白板 PPT

> 一套完整的流水线，把任意一句话主题，变成一整套白板手绘风格的 PPT 图片。

---

## 核心思路

制作 PPT 最难的不是排版，而是**把混乱的想法变成清晰的结构**。

这个项目探索了一条完整的路径：

```
一句话主题
    ↓
大纲（章节 + 要点）
    ↓
完整文档（数据、案例、细节）
    ↓
每页分页内容（标题 + 要点 + 视觉建议）
    ↓
每页图片提示词（布局区域 + 颜色 + 图标描述）
    ↓
白板手绘风格图片（AI 生成）
```

每一步都有明确的输入和输出，前一步的产出直接喂给下一步。最终你拿到的是一套可以直接用于演示的图片，风格统一、内容清晰。

## 四步流水线详解

### Step 1：大纲

从一句话主题提炼出结构化大纲——章节划分 + 每章核心要点。

这一步回答的问题是：**这个主题应该讲哪几件事？顺序是什么？**

### Step 2：完整文档

将大纲丰富为完整内容——补充真实数据、具体案例、关键细节。

这一步回答的问题是：**每件事的支撑论据是什么？听众需要知道哪些具体信息？**

### Step 3：分页内容

按 PPT 页拆分，每页包含：页码、标题、3-5 个要点、视觉建议。

这一步回答的问题是：**每一页放什么？用什么形式呈现最清晰？**

### Step 4：图片提示词

为每页生成白板手绘风格的图片提示词，精确描述：
- 布局区域划分（标题区 / 内容区 / 底部区）
- 颜色方案和图标描述
- 手绘元素（箭头、图框、连线）
- 文字内容（直接嵌入提示词）

这一步的产出可以直接喂给图像生成模型。

## 图片风格

所有图片使用统一的白板手绘风格：

- 干净白色白板背景
- 类 Excalidraw 手绘线条
- 彩色马克笔笔触
- 中文手写标签
- 简笔图标（无写实元素）

**风格底板（每次复用）：**

```
Style: clean white whiteboard background, hand-drawn sketch like Excalidraw,
thick colorful marker strokes, casual handwritten Chinese labels,
simple doodle icons, wobbly hand-drawn arrows. No photorealistic elements.
```

## 如何生成图片

```bash
# 从提示词文件生成图片（推荐）
python -X utf8 py/generate_image.py \
  --prompt-file test-cases/介绍深圳/prompt-p01.txt \
  --output test-cases/介绍深圳/p01-封面.jpg \
  --aspect-ratio 16:9 \
  --image-size 4K

# 直接传入提示词
python -X utf8 py/generate_image.py \
  --prompt "ALL TEXT MUST BE IN CHINESE. ..." \
  --output output.jpg
```

API Key 配置：在项目根目录创建 `api.md` 文件，填入你的 [Nano Banana Pro](https://nanobananapro.com) API Key。

## 设计原则

**一句话概括**：用 AI 做 PPT，关键不是让 AI 直接生成 PPT，而是让 AI 帮你把想法结构化，然后把结构化的内容转化为图片。

几个关键选择：

1. **分步而非一步到位**：每步聚焦一件事，比一步到位的成功率高得多
2. **白板手绘而非精美模板**：手绘风格容错率高，AI 生成的不一致感反而成了风格
3. **提示词精确描述布局**：不是描述"好看的图"，而是描述具体的空间划分和元素位置
4. **ALL TEXT MUST BE IN CHINESE**：图像生成模型默认倾向英文，必须显式指定
